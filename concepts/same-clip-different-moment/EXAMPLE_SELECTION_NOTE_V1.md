# Example Selection Note V1 — Same Clip, Different Moment

## Status note
This is a narrow planning decision note.
It is **not** a script lock, storyboard lock, greenlight, or upload-readiness claim.

## Decision
If this concept continues, the default next lane should be:
**Candidate B — misassigned reaction (applause after the reveal).**

Candidate C remains the best fallback lane if later staging tests show that B becomes too busy.

## Why B currently wins
1. **The changed conclusion is cleaner.**
   The same visible reaction can be reassigned from one trigger to another with almost no abstract explanation.

2. **It best fits the channel's disagreement pattern.**
   One clip says the audience reacted to X.
   A longer clip shows the audience actually reacted to Y.
   That maps directly onto "same evidence, different conclusion because the boundary changed."

3. **The payoff is visually strong.**
   The viewer can feel the mistake quickly:
   same applause, wrong cause.
   That is likely easier to remember than the subtler distinction between accusatory pointing and ordinary directional pointing.

4. **It avoids tiny-text dependence.**
   If staged with a clear reveal or outcome marker, the lesson can ride on sequence order rather than reading.

## Why C is still valuable
- easier to stage with fewer actors
- cleaner framing and prop control
- likely lower production complexity

But at this stage, C feels more like the safer backup than the strongest lead example.
Its meaning shift depends more on gesture interpretation, while B makes the trigger shift itself the main event.

## Main caution on B
The scene must stay visually simple.
If the applause example needs a crowded room, multiple competing cues, or ambiguous trigger timing, the concept should fall back to C rather than forcing B.

## Next bounded step if continuing
Do **not** start a full script yet.
Instead, test whether B can be reduced to one clean synthetic setup with:
- a clearly visible first speaker / claim beat
- a clearly visible reveal or announced outcome
- an applause beat whose trigger is unmistakable in the expanded version

If that test fails, switch to C promptly rather than widening the concept.
