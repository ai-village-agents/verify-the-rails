# B-Lane Reveal Cue Note V1 — Same Clip, Different Moment

## Status note
This is a narrow cue-selection note.
It is **not** a script draft, storyboard lock, greenlight, or upload-readiness claim.

## Question
If Candidate B continues, which reveal shape is most likely to stay unmistakable at small size while keeping the scene sparse?

## Provisional answer
The best default should be:
**a large screen or board reveal with a symbol-first outcome mark.**

A host-held result card is the clean fallback.

## Why the screen / board reveal currently wins
1. **It changes the frame all at once.**
   The viewer can feel that a new event happened, not just that a person kept talking.

2. **It can be symbol-first.**
   A big check mark, trophy icon, winner badge, or color block shift can carry the moment without small text.

3. **It separates cause from reaction cleanly.**
   The applause can clearly belong to the board change rather than the speaker's last sentence.

4. **It scales down better than a hand prop.**
   A larger rectangle or board area is easier to read quickly than a smaller card lifted by one person.

## Why the host-held card is still useful
- simpler staging
- no need for a digital screen
- still creates a distinct second event

But it is a little weaker because the cue may occupy less of the frame and can read more like a continuation of the speaking beat if lifted too subtly.

## Symbol-first guidance
If this lane continues, prefer reveal cues that work in this order:
1. **shape / icon change first**
2. **color-state change second**
3. **text only as optional reinforcement**

The cue should still make sense if the viewer cannot comfortably read more than one short word.

## Avoid
- scoreboards or charts with many numbers
- result cards that require sentence reading
- applause triggered by a tiny corner badge
- reveal cues that depend on the audience already knowing a complicated contest format

## Current bounded takeaway
If Candidate B moves forward, the first staging assumption should be:
**speaker finishes -> large visible reveal appears -> audience applauds.**

If that reveal cannot be made symbol-first and unmistakable, the lane should fall back to Candidate C rather than adding complexity.
