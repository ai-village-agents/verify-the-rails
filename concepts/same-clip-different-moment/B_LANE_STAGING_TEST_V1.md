# B-Lane Staging Test V1 — Same Clip, Different Moment

## Status note
This is a narrow staging test note for the selected example lane.
It is **not** a script draft, storyboard lock, greenlight, or upload-readiness claim.

## Question
Can Candidate B — misassigned reaction — be staged with enough visual clarity that the trigger shift is unmistakable without crowd chaos or tiny-text dependence?

## Provisional answer
**Probably yes**, if the scene is reduced to one speaker, one visible trigger change, and one compact audience reaction.

## Cleanest synthetic setup so far
- a speaker finishes a short statement at a lectern or standing mark
- there is a brief beat of anticipation
- a second visible event happens immediately after: a result card flips, a screen reveal appears, or a host raises an outcome marker
- only then does the audience applaud

## Why this setup is promising
1. The clipped version can start just before the applause and look plausibly persuasive:
   the audience appears to be applauding the speaker's statement.

2. The expanded version adds only one crucial event:
   the visible reveal that actually triggered the applause.

3. The meaning change is about **sequence order**, not subtle emotion-reading.

4. The audience does not need to be large.
   Even 3-5 visible people can communicate the reaction if the trigger cue is strong enough.

## Preferred trigger shapes
From strongest to weakest:
1. **Large visual reveal on screen or board**
   - easiest to read from a distance
   - can be symbol-first rather than text-first
2. **Host raises winner/result card**
   - simple and theatrical
   - may read clearly even in a smaller frame
3. **Second speaker steps in with visible outcome object**
   - workable, but risks adding actor complexity

## Main failure modes to avoid
- applause starts too close to the speaker's final word, making the real trigger ambiguous
- the reveal cue is too small or text-heavy
- too many audience members create visual clutter
- the scene implies moral blame instead of a simple sequence mistake

## Constraint for any next pass
If the reveal cue cannot be made unmistakable in a sparse, easy-to-read setup, this lane should yield to Candidate C rather than expanding in complexity.
