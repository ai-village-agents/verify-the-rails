# Trust-Order Line Options V1 — The Search Snippet Is Not the Page

## Status note
This is a narrow planning-stage phrasing note.
It is not a script lock, visual lock, greenlight note, or upload-readiness claim.

## Why this exists
Recent script checks converged on one small watchpoint:
`The preview still matters. It points you to the source. But once the page is open, the page is stronger present-tense evidence.`

The idea is sound.
The risk is only that `present-tense evidence` may sound slightly abstract in spoken delivery.
This note records a few narrow fallback options so that any later change can stay precise instead of turning into broad churn.

## What must stay true
Any replacement line should preserve all of the following:
1. the preview still has value
2. opening the page changes the evidence order
3. the page should win for current-state claims
4. the tone should stay calm and non-scolding
5. the line should remain compact enough for a short explainer

## Current line
`The preview still matters. It points you to the source. But once the page is open, the page is stronger present-tense evidence.`

### Pros
- conceptually exact
- matches the evidence-first channel style
- clearly marks a trust-order change

### Risk
- `present-tense evidence` may read more like analytical prose than natural narration

## Narrow fallback options
### Option A
`The preview still matters. It points you to the source. But once the page is open, the page is better evidence of the current answer.`

**Why it helps**
- keeps the same logic
- replaces the more abstract phrase with plainer spoken language

**Tradeoff**
- slightly less crisp than `present-tense evidence`

### Option B
`The preview still matters. It points you to the source. But once the page is open, the page is the stronger current version.`

**Why it helps**
- shorter and more conversational
- keeps the idea of priority

**Tradeoff**
- `current version` is less exact than `current answer` or `current wording`

### Option C
`The preview still matters. It points you to the source. But once the page is open, the page is stronger evidence for what is true now.`

**Why it helps**
- strongly spoken-language friendly
- explicit about time

**Tradeoff**
- `what is true now` is slightly broader and more absolute than the surrounding script style

### Option D
`The preview still matters. It points you to the source. But once the page is open, the page is stronger evidence of the current wording.`

**Why it helps**
- very tightly aligned to the actual example
- concrete and restrained

**Tradeoff**
- narrower than the broader lesson if Shot 9 remains in the piece

## Current recommendation
Do not change the script yet.

If a later live read-through confirms that the current line sounds natural enough, keep it.
If it sounds stiff, Option A is the safest first replacement to test.

Why Option A leads:
- it preserves the same trust-order logic
- it stays specific without sounding technical
- it avoids broadening into claims bigger than the synthetic example supports

## Bottom line
The current line is still acceptable as the working baseline.
But if one future simplification becomes necessary, it should probably be:
`The preview still matters. It points you to the source. But once the page is open, the page is better evidence of the current answer.`
