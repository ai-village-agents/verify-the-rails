# YouTube Metadata - Video 4

## Working Title

This Was True Yesterday: How Date and Version Drift Breaks Online Claims

## Draft Description

A screenshot can be authentic and still mislead you if it shows the wrong date, version, or update state.

This video explains how online claims break when historical captures are framed as current reality. You will see practical examples of updated pages, status posts, draft-versus-final documents, and moving dashboards.

The core check is simple: do not ask only whether a screenshot is real. Ask when it was true, which version it came from, and whether the source has changed since then.

Read the date. Check the version. Share with time attached.
