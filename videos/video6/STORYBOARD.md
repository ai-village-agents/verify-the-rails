# Storyboard - Video 6

## Visual Language

- same Verify the Rails editorial style as Videos 1-5
- cool blue for observed facts, amber for missing context, muted red for overreach
- recurring labels: `Observed`, `Inference`, `Unknown`, `Count`, `Rate`
- recurring prompt line: `Real number, incomplete conclusion.`

## Shots

1. **01_title_card.png**
   - Visual goal: set thesis immediately with calm authority
   - On-screen text: `A Real Number, Incomplete Conclusion` / `Missing denominator, baseline, or window can mislead`
   - Narration summary: opens on the idea that real numbers can still support wrong conclusions

2. **02_big_number_hook.png**
   - Visual goal: create tension between dramatic count and interpretation risk
   - On-screen text: `24,000 reports filed` / `Real number` / `But what does it mean?`
   - Narration summary: introduces evidence-first framing and the need for context rails

3. **03_out_of_how_many.png**
   - Visual goal: make denominator question concrete and memorable
   - On-screen text: `Out of how many?` / `24,000 out of 18,000,000 = 1.3 per 1,000`
   - Narration summary: first worked example shows large count without denominator

4. **04_count_vs_rate.png**
   - Visual goal: distinguish raw count claims from rate/prevalence claims
   - On-screen text: `Count answers volume` / `Rate answers prevalence` / `Use the metric that matches the claim`
   - Narration summary: explains why count and rate support different conclusions

5. **05_small_base_rate.png**
   - Visual goal: show how percent growth can look huge from tiny base
   - On-screen text: `"Up 100%"` / `3 -> 6 cases` / `Correct math, limited scope`
   - Narration summary: second worked example on doubled percentage from a tiny base

6. **06_time_window_reveal.png**
   - Visual goal: expose how long windows can inflate short-term impressions
   - On-screen text: `1,200 incidents` / `Over 5 years` / `~20 per month average`
   - Narration summary: fourth worked example on totals spread over long time windows

7. **07_rank_vs_population.png**
   - Visual goal: show city ranking reversal after per-capita adjustment
   - On-screen text: `City A: 12,000 complaints (4M people)` / `City B: 8,000 complaints (1M people)` / `Per 1,000: A=3, B=8`
   - Narration summary: third worked example on city comparison without population adjustment

8. **08_baseline_shift.png**
   - Visual goal: anchor baseline check as separate from denominator
   - On-screen text: `Compared with what baseline?` / `Last month? Last year? Typical range?`
   - Narration summary: reframes interpretation around baseline choice and trend context

9. **09_claim_limit_template.png**
   - Visual goal: teach claim-boundary language viewers can reuse
   - On-screen text: `Observed:` / `Inference:` / `Unknown:` / `Claim limit`
   - Narration summary: models disciplined separation between direct evidence and interpretation

10. **10_fast_verification_steps.png**
    - Visual goal: present a fast reusable checklist for daily use
    - On-screen text: `1 Out of how many?` / `2 Compared with what baseline?` / `3 Over what time window?` / `4 Count or rate?`
    - Narration summary: delivers four-question verification routine and uncertainty discipline

11. **11_observed_inference_template.png**
    - Visual goal: reinforce observed-vs-inference note format
    - On-screen text: `Observed: source says X` / `Inference: may imply Y` / `Unknown: still need Z`
    - Narration summary: emphasizes evidence-first notation before sharing claims

12. **12_closing_card.png**
    - Visual goal: close with memorable line tied to channel identity
    - On-screen text: `Verify the Rails` / `Real number, incomplete conclusion.` / `Check denominator, baseline, window`
    - Narration summary: final reminder to verify context before amplifying a number claim
