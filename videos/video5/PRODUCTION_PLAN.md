# Production Plan - Video 5

## Goal

Produce a 5-6 minute explainer showing how genuine screenshots can mislead when crucial context sits just outside the crop.

## Research / Evidence Needs

- one chart or dashboard example where a hidden filter changes interpretation
- one status/update example where a banner or later update changes meaning
- one document example where the hidden header/footer changes status
- one source-authenticity example where hidden domain or URL context matters

## Script Structure

1. **Hook (0:00-0:35):** same screenshot seems decisive until the uncropped view appears.
2. **Reframe (0:35-1:05):** ask not only `is it real?` but also `what is outside the frame?`
3. **Why Crops Mislead (1:05-2:00):** crops remove context rails such as date, domain, filters, and labels.
4. **Worked Examples (2:00-4:20):** chart filter, status banner, draft header, mirror domain.
5. **Verification Routine (4:20-5:15):** recover missing context before sharing.
6. **Close (5:15-5:45):** screenshot true, conclusion overstated.

## Visual Approach

- start with tight crops, then zoom out into full-frame context cards
- use bounding boxes for hidden clues that appear only in the uncropped version
- persistent labels: `Inside the crop`, `Outside the crop`, `Observed`, `Inference`
- cool blue for direct evidence, amber for caution, muted red for overreach

## Narration Approach

- plain-language explanation of what a screenshot can and cannot prove by itself
- repeatedly separate visible evidence from missing evidence
- emphasize that hidden context can weaken confidence without proving bad intent
