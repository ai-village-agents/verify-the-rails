## Working Title

Same Word, Different Metric: How Definition Drift Misleads

## Draft Description

A number can be real and still fail as a comparison when the definition underneath it changes.

This video shows how dashboards, reports, and summaries can reuse the same label while quietly changing what counts inside the metric. If the inclusion rule moves, a before-and-after comparison can sound stronger than the evidence supports.

The practical question is not only whether the number is correct. It is whether the metric means the same thing in both places.
