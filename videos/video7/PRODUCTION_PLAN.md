# Production Plan - Video 7

## Goal

Produce a 4-6 minute explainer showing how a metric can become misleading when the definition changes but the label appears unchanged.

## Research / Evidence Needs

- one example where the same dashboard label covers different inclusion rules over time
- one example where a category expands and the count rises without a real-world surge of the same size
- one example where a summary compares unlike periods because the measurement definition shifted
- one practical routine for checking notes, methodology, changelogs, or footnotes before comparing numbers

## Script Structure

1. **Hook (0:00-0:30):** one metric appears to jump dramatically.
2. **Reframe (0:30-1:00):** ask `did reality change, or did the definition change?`
3. **Definition Drift (1:00-2:00):** same label, different inclusion rule.
4. **Worked Example 1 (2:00-3:00):** category expansion inflates the count.
5. **Worked Example 2 (3:00-4:00):** dashboard comparison breaks across a methodology note.
6. **Verification Routine (4:00-4:45):** find the metric definition, inclusion rule, and change date.
7. **Close (4:45-5:10):** same word, different metric.

## Visual Approach

- dashboard cards that keep the same headline but swap the counting rule beneath
- comparison tables with `included before` and `included now` rails
- callouts for methodology notes, footnotes, and definition changelogs
- blue for observed facts, amber for interpretation limits, muted red for overclaiming

## Narration Approach

- emphasize that the number may be honestly reported while still being non-comparable
- use plain language like `what got counted changed`
- keep the lesson practical and repeatable for non-experts
