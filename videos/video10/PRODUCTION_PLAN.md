# Production Plan - Video 10

## Goal

Produce a 4-6 minute explainer showing how chart framing choices can magnify, flatten, or redirect interpretation without changing the underlying data.

## Research / Evidence Needs

- one example of a truncated or compressed axis changing perceived magnitude
- one example of time-window or scale choice changing perceived trend
- one example where cumulative display creates a misleading sense of acceleration
- one practical checklist for reading charts before sharing them

## Script Structure

1. **Hook:** two charts feel different even before you inspect the numbers.
2. **Reframe:** real data does not guarantee fair visual impression.
3. **Mechanism:** axis, scale, and framing choices shape perception.
4. **Worked Example 1:** truncated axis exaggeration.
5. **Worked Example 2:** time window / scale framing.
6. **Worked Example 3:** cumulative chart versus rate chart.
7. **Verification Routine:** inspect axes, baselines, windows, and chart type.
8. **Close:** real data can still support a distorted visual impression.

## Visual Approach

- paired charts with identical data but different axis treatments
- explicit rails for `Observed / Impression / Unknown`
- callouts on zero baseline, window length, and chart type
- same Verify the Rails dark editorial style with high-contrast highlights

## Narration Approach

- calm and non-accusatory
- emphasize that charts are not automatically deceptive, but design choices matter
- keep the boundary between data integrity and interpretation integrity explicit
