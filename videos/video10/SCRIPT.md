# Script - Video 10

## Public/Working Title

One Chart Is Not the Whole Story: How Axis and Framing Choices Mislead.

## Promise

A chart can use accurate numbers and still create a misleading impression.
This episode shows how axis range, time window, and cumulative framing can change what viewers feel they are seeing.

## Core Lens

Use this reusable check on every chart:

- `Observed`: what the chart directly shows
- `Impression`: what the chart design makes you feel is happening
- `Unknown`: what context is still missing

## Example 1: Truncated Y-Axis

Two bars differ by a small amount.
One chart starts the y-axis near the measured values instead of zero.
The visible gap looks huge, even though the absolute change is modest.

## Example 2: Same Series, Different Framing

The exact same data is plotted twice.
Version A uses a short time window and a tight vertical range, which feels urgent.
Version B uses a longer window or wider scale, which feels steadier.

## Example 3: Cumulative Curve vs Rate

A cumulative total line keeps rising and often curves upward.
Viewers can read this as acceleration.
But the period-to-period rate can be roughly flat.
The cumulative line mainly shows accumulation over time.

## Practical Verification Checklist

1. Check the y-axis baseline and range.
2. Check the time window and ask what was excluded.
3. Check whether scale choices changed visual steepness.
4. Check whether the chart is cumulative, rate, or both.
5. Write one `Observed / Impression / Unknown` note before sharing.

## Close

Real data can still support a distorted visual impression.
